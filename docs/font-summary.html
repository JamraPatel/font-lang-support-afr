Google Fonts Pan-African Latin Gap Analysis

Font Level Results

This report provides a breakdown of the Pan-African Latin support errors for the Google Fonts Library.

Usage

  • The main screen shows the error report by font family. The report can be filtered alphabetically using the options in the side bar.
  • Selecting a font family will shows the cumulative errors in the side bar and provides an option to expand the family.
  • Once an individual font is selected a breakdown of errors by language tag is available by pressing the breakdown button.
  • Selecting any row in the chart will dump the errors in the side bar. Clicking on error items in the legend will toggle them on/off in the chart.
